Admin users


You can add colleagues to the admin console by creating a new user.

There are 2 levels of access:

  • Regular Admins: Can access the admin console to view the Dashboard, User and Application tabs, and sync unsynced data.
  • Super Admins: Have all Regular Admin privileges, plus the ability to add or remove admins, create Super Admins, and control feature flags.
